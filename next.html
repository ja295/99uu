HTML Elements

Headings

Use <h1></h1> to <h6></h6> for headings. h1 is the most important, and h6 is the least.

Paragraphs

Use <p></p> to create a paragraph. This is used for blocks of text.

Links

Use <a href="url"></a> to create hyperlinks. For example, this is a link.

Images

Use <img src="image-url" alt="description"> to include images in your HTML.

Lists

Use <ul></ul> for unordered lists and <ol></ol> for ordered lists.

CSS Properties

Color

Use color to set text colors (e.g., color: blue;).

Background Color

Use background-color for element backgrounds (e.g., background-color: lightblue;).

Margins and Padding

Use margin and padding to create space around elements.

Font Size

Set font sizes with font-size. Use vw for responsive typography (e.g., font-size: 2vw;).

Display

Control how elements are rendered (display: block;, display: inline;, etc.).

Flexbox

Use Flexbox for layout with display: flex; to create flexible and responsive layouts.

Grid

Use CSS Grid for more complex layouts with display: grid;.
